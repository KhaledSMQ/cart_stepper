# Cart Stepper Example

A comprehensive demo app showcasing all features of the `cart_stepper` package.

## Features Demonstrated

The example app includes 6 tabs:

1. **Basics** - Size variants, add-to-cart button styles, and basic functionality
2. **Styles** - Pre-built themes, custom styling, and animation presets
3. **Async** - Loading indicators, optimistic updates, debounce mode, and error handling
4. **Advanced** - Min/max limits, long press, quantity formatters, callbacks, validation, auto-collapse, and manual input
5. **Components** - Controller usage, CartBadge, CartStepperGroup, theming, and standalone widgets
6. **Real World** - A complete shopping cart implementation

## Running the Example

```bash
cd example
flutter create --platforms=android .   # or ios, web, macos, etc.
flutter run
```
